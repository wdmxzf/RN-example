# RN-example
RN 项目构建
## 基础环境
* Node（>14）
* JDK（jdk11）
* Android 环境或者 iOS 环境
* npm 、yarn
* 设置镜像，但是不要 使用 **cnpm** 命令，它会导致package 不识别路径。
